 ## CCE109L - Fundamentals of Java Programming

 Welcome to my **Java Programming Fundamentals** repository! This repository contains my practice excercises, mini projects, and exam preparations from my first year as a Computer Science student. It serves as a collection of my learning journey in mastering the basics of java.


 ## What you'll find here

 - **Practice Excercises:** Small programs to reinforce concepts like loops, conditionals, arrays, methods, and (write to file) file handling.
 - **Projects:** Mini-projects that integrate multiple concepts to build practical applications.
 - **Exam Preparations:** Sample problems and solutions to help review key Java fundamentals.

 ## Topics covered
 - Variables, data types, and operators
 - Control flow (if-else, switch-case, loops)
 - Methods
 - Arrays and Strings
 - Basic input/output (scanner, print to console)
